# raven-game-editor

Small keyboard synth written in Rust using
[midir](https://crates.io/crates/midir/) and
[egui](https://crates.io/crates/egui). There's no sound sound
implemented yet, it currently only reads MIDI events from a selected
MIDI input and shows the pressed keys.
